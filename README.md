# TaskBoard App

React、TypeScript、および Supabase を使用して開発した、タスク管理アプリケーションです。

## プロジェクトについて
このプロジェクトは、フロントエンド開発とバックエンド連携の**学習用**として作成されました。
以下の技術スタックの基礎を習得することを目的としています。

- **React & TypeScript**: 型安全な UI コンポーネントの開発。
- **Supabase**: 認証（Authentication）およびデータベース（PostgreSQL）の実装。
- **Tailwind CSS**: ユーティリティファーストなスタイリングによるレスポンシブデザイン。
- **Vite**: 高速なビルド環境の構築。

## 主な機能

- **ユーザー認証**: Supabase Auth を利用したサインアップ、ログイン、ログアウト機能。
- **タスク管理 (CRUD)**:
  - タスクの作成（タイトル、詳細、ステータス、優先度）。
  - タスクの一覧表示（リアルタイムな状態反映）。
  - タスクの編集（インライン編集モード）。
  - タスクの削除。
- **レスポンシブデザイン**: ダークモードを基調とした、PC・スマホ両対応の UI。

## 使用技術

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **Backend/DB**: Supabase (Auth, PostgreSQL)
